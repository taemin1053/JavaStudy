# 자바 스터디 (Java Study)

- 자바 기초 문법과 객체지향 원리를 학습하기 위한 저장소입니다.

### 참고 강의
 - 인프런 김영한 자바 기초 
### 🛠 사용 기술 
- Java 21
- IntelliJ IDEA
- Git/GitHub
### 파일 위치
 - src/
### 각 package설명 
 - class1 = 클래스
 - ref = 메소드
 - oop = 객체 지향 프로그래맹
 - construct = 생성자
 - com.helloshop, pack = 패키지 
 - access = 접근자
 - static1,static2 = 자바 구조, 정적변수
